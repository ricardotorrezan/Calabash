Comandos básicos git:

```ruby
git pull - Tras a branchconteudo de algum lugar para outro lugar:
git pull
git pull origin master - /tras o conteúdo da branch master (origin)


git clone - Clona o repositorio:
git clone https://github.com/thiagomarquessp/calabash_aula_1.git

git checkout -b "NOME-BRANCH" cria uma nova branch:
git checkout -b "TP-Aula-1"

git status verifica o momento atual da sua branch:
git status


git add + arquivo adiciona arquivo a memória (antes de subir)
git add /automacao/automacao/calabash/calabash_aula_1/arquivo.#!/usr/bin/env ruby -wKU


git commit -m "Mensagem" faz commit (Coloque a mensagem)
git commit -m "Esse é um exemplo dos comandos básicos do git"



git push - empurra para o repositorio
git push

```
